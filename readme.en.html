Overview

Ut Video Codec Suite is a multi-platform and multi-interface lossless video codec.

Environment

Minimum requirement (Windows x86)

OS
Windows Vista or later
CPU
i686-compatible CPU with SSE2 support
(e.g. Pentium 4 or later)

Minimum requirement (Windows x64)

OS
Windows Vista (x64) or later
CPU
CPU with AMD64 or Intel 64 (formerly EM64T) support

Minimum requirement (Mac OS X)

OS
Mac OS X 10.6 Snow Leopard or later
CPU
i686-compatible CPU with SSE2 support
(All Intel Mac meet this requirements)

Recommended

CPU
1st Generation Intel Core i series (codename "Nehalem") or later

Installation Notice

Windows

If version 10.2.0 or older is installed, you have to uninstall that version before you install this version.

If VCM codec is not registered properly, merge utvideo.reg in install destination with registry as a last resort. This file assumes Windows directory is C:\Windows.
